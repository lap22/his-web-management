# HIS Web Client (React + Vite + TypeScript)

Minimal starter for a Hospital Information System (HIS) web client.

## Setup

Install dependencies:

```bash
cd his-web-client
npm install
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

Environment variables: copy `.env.example` to `.env` and set `VITE_API_URL`.
